Data explanation

The dataset is a mix of data from the Paleobiology Database (accessible via Fossilworks or PaleobioDB), FosFARbase and lots of literature research. It contains coordinates, count data and also factorial variables.

Variable definitions

variable name details
species the binomial species name
order the taxonomic order of the species
ma_range duration the species covered in the fossil record in million years
mni_max minimum number of individuals in one location, maximum value for each species
spec_max maximum number of specimens in one location, maximum value for each species
grid_cells number of grid cells a species occupies
gcd maximum great circle distance; shortest connection between two points on the surface of a sphere
lat_range maximum latitudinal range of a species (maximum difference between paleolatitudes)
mean_lat mean latitude calculated from all species occurrences
min_lat minimum latitude
hab.cat.b habitat category; low, high, or highlow (last one meaning both). Distinguished basically between flowing and stagnant water bodies
habitat_width number of habitats utilized by the species
bodysize maximum or only available skull length; values scaled from 0 to 1
noo total number of occurrences per species

The median durations between habitat width groups (1,2,3) differ significantly, although category 4 just has one single count and therefore cannot be considered. Removing all singe-interval species does not influence this pattern. There is a positive correlation (apparently non monotonous as stronger using spearmans rank correlation) between number of occurrences and number of habitat types. The resason is obvious to some degree, as the number of habitats is required as minimum number of occurrences for each habitat width. Regarding preservation probability, the number of occurrences simply could be the reason for the observed pattern between duration and habitat width as it potentially influences both duration and habitat width. However, correcting the habitat width for the number of occurrences in each species does not seem justified as this would exclude the possibility of species just being more abundant as well as it would excessively punish very abundant species. E.g. Opisthotriton kayi with 71 occurrences in 3 habitat types would end up having a lower habitat width index than a species occurring in just one habitat with 3 occurrenes. Being rare would let species appear to occupy in more habitats.

Might these differences be caused by larger sample size in the northern hemisphere?

If habitat width appears larger with bigger latitudes (north), this might be a result of a more comprehensive fossil record in the northern hemisphere. There is two factors: more occurrences in the north, and also more species in the north. Recalculating the habitat widths for the northern hemisphere species, using the same amount of occurrences like found in the southern hemisphere, then correlating the new habitat width again with mean latitude, is likely to show if sampling or preservation bias causes the effect. Based on the occurrence data underlying the species based dataset, I resampled the data and calculated for each set (500) the habitat width for each species, this way simluating a dataset with equally distributed occurrences between northern and southern hemisphere.

Also: there might be some deviation from constant variance assumption (residuals~fitted values), there seem to be 3 outliers in the data (normal-q-q plot: 9,61, 146). Residuals are getting larger with values getting larger, with one clear outlier. ### Mulitcolinearity min_lat and mean_lat are strongly correlated, so assumptions are violated here. According to wikipedia, “Multicollinearity does not reduce the predictive power or reliability of the model as a whole, at least within the sample data set; it only affects calculations regarding individual predictors. That is, a multiple regression model with correlated predictors can indicate how well the entire bundle of predictors predicts the outcome variable, but it may not give valid results about any individual predictor, or about which predictors are redundant with respect to others”. This is especially strange as removing one of the correlated variables results in a stark drop of R squared. For colinerarity, correlation needs to be linear. Coefficients change not dramatically when removing one of the variables, but testing the insignificant variables from the multilinear model alone shows significant effects in simple linear regression for noo and habitat_width. I will keep both variables for now.

Cross validation of the linear model with logged response variable shows that high values are constantly underpredicted, while small values tend to get overpredicted. However, the trend within the predictions is right: Large values are mostly predicted larger values by the model than small values. Cross validated standard error of estimate is 4.483 million years in this model.

Conclusion

The linear model explains around 30% of the observed data, however it suffers from multicolinearity between mean and minimum latitude. This makes estimates for predictor coeffiecients less precise and potentially üroduces type 2 errors. Removing one of these factors however reduces explained variance dramatically. Logging the dependent variable seems appropriate, given that the outcome variable is highly skewed.
